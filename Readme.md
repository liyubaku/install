使用root用户登录，运行以下命令：

wget --no-check-certificate -O shadowsocks-libev.sh https://github.com/liyubaku/install/raw/master/shadowsocks-libev.sh
chmod +x shadowsocks-libev.sh
./shadowsocks-libev.sh 2>&1 | tee shadowsocks-libev.log
